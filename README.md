# RAG-Document-QandA-with-NVIDIA-NIM
A RAG (Retrieval-Augmented Generation) application built with Streamlit and NVIDIA AI endpoints. Features PDF document processing, FAISS vector storage, and LLM-powered Q&amp;A using Meta's Llama3-70B model. Tech stack: Streamlit, LangChain, NVIDIA AI endpoints, FAISS, HuggingFace Embeddings and Python for efficient document retrieval.
